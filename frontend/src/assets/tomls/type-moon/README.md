# type-moon — Fate 圣杯战争模拟器

一个基于 Fate 系列（Nasuverse）的圣杯战争模拟器，玩家扮演从者参与圣杯战争。

## 特色

- **从者视角**：玩家扮演从者而非御主，体验英灵的战斗与抉择
- **经过微调的行为约束**：避免常见的 AI 逻辑问题（御主试图"建立支配"、不理解从者能力、过度强调神秘隐匿等）
- **详细的状态栏**：追踪从者参数、技能、宝具、物品栏、人际关系、御主状态
- **支持多次圣杯战争**：第四次（1994）、第五次（2004）及自定义战争

## 文件列表

| 文件 | 类型 | 说明 |
| --- | --- | --- |
| `fate.simulator.chr.toml` | Simulator | 主模拟器：定义 Fate 世界观、圣杯战争规则、从者/御主机制、状态栏格式、行为准则 |

### Addon 扩展包

以下 Addon 提供具体圣杯战争的参战者数据，按需加载：

| 文件 | 名称 | 说明 |
| --- | --- | --- |
| `hgw4.addon.chr.toml` | The 4th Holy Grail War | 第四次圣杯战争（1994）：全部七组 Master/Servant 的详细设定，包括参数、技能、宝具、性格 |
| `hgw5.addon.chr.toml` | The 5th Holy Grail War | 第五次圣杯战争（2004）：全部参战者设定，含伪 Assassin、真 Assassin 及残留的吉尔伽美什 |

## 使用方式

1. 载入 `fate.simulator.chr.toml` 作为模拟器
2. 按需载入 `hgw4.addon.chr.toml` 或 `hgw5.addon.chr.toml`（根据你想参与的战争）
3. 载入玩家角色卡（可使用 `blank/example.player.chr.toml` 模板创建）
4. 按照开场白中的模板填写从者信息，开始游戏

## 注意事项

- 不建议选择远坂时臣作为御主（详见模拟器内说明）
- 不建议发展复杂感情或"补魔"剧情（缺少对应状态栏支持）
- 建议将从者技能和宝具的详细设定写在个人设定或印象记忆中，确保模拟器始终能读取
