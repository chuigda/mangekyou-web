# blank — 空白模板

这是一个最小化的 CHR 文件模板集，包含所有可用字段及其注释说明，适合作为创建新世界的起点。

## 文件列表

| 文件 | 类型 | 说明 |
| --- | --- | --- |
| `simulator.chr.toml` | Simulator | 模拟器模板，包含世界名称、开场白、状态栏格式、指令定义、世界设定等全部字段 |
| `player.chr.toml` | Player | 玩家角色卡模板，包含角色名称和设定字段 |
| `addon.chr.toml` | Addon | 扩展包模板，包含所有可追加的字段（世界设定、角色数据库、状态栏、记忆规则等） |

## 使用方式

1. 复制本文件夹中的模板文件
2. 根据注释填写你需要的字段，不需要的字段可以删除或保持注释状态
3. 在 Mangekyou 界面中分别上传 Simulator、Player 和 Addon 文件
